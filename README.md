# 卦馆网站

这是一个基于命理、风水、阴阳宅主题的网站。

## 部署说明

### 部署到 GitHub Pages

1. 将本代码包中的所有文件上传到您的 GitHub 仓库：`https://github.com/empirewg/sunshine`

2. 确保文件结构如下：
   ```
   /
   ├── index.html          # 首页
   ├── shoping.html        # 风水器物页面
   ├── README.md           # 说明文件
   └── images/             # 图片文件夹
       ├── 产品-大象.jpg
       ├── 产品-孔雀.jpg
       ├── 产品-葫芦.jpg
       ├── 产品-聚宝盆.jpg
       ├── 产品-铜钱葫芦.jpg
       └── 产品-罗盘.jpg
   ```

3. 在 GitHub 仓库设置中启用 GitHub Pages：
   - 进入仓库的 Settings -> Pages
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main" 或 "master"，文件夹选择 "/ (root)"
   - 点击 Save

4. 等待几分钟后，网站将通过 `https://empirewg.github.io/sunshine/` 访问
